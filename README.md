# Voqal iOS SDK

The voice-first assistant SDK for iOS, distributed as a binary XCFramework via
Swift Package Manager.

## Install

In Xcode: **File → Add Package Dependencies…** and enter:

```
https://github.com/VoqalAI/voqal-ios
```

Pick **Up to Next Major** from **1.0.0**, add the **VoqalSDK** library.

- Minimum iOS: **16**
- You need a Voqal API key (`pk_live_…`) — set `config.apiKey`.
